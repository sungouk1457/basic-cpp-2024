## 1일차
- 개발환경 설치 
	- Visual Studio 2022 Community 설치
	- C, C++ 소스코드 실행
- C++ 기초	
	- 자료형
		- 문자:문자(char)/문자열
		- 숫자:정수(int)/실수
	- 선택제어문:if문,switch
	- 반복제어문:for문,while문

## 2일차
- C++기초
	- 매개변수
	- 인라인(inline)함수
		- 프로그램 코드라인 안으로 들어가 버린 함수
	- 이름공간
	- C에서 메모리
		1. 스택 : 지역변수,매개변수
		2. heap : 사용자에게 할당된 영역
			- 사용을 할려면 malloc()(c++:new)으로 할당을 받는다-free()(c++:delete)로 반환
		3. data : 함수, 전역변수, 상수, 문자리터럴
	- 참조자
